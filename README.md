## ğŸ¨ Customization

### Adding New Destinations

1. Copy an existing destination section in `index.html`
2. Update the content, dates, and navigation links
3. Add corresponding navigation item to the header menu
4. Update the JavaScript arrays if needed

### Changing Colors

The main color scheme can be modified in `styles.css`:
- Primary color: `#e74c3c` (red accent)
- Secondary color: `#2c3e50` (dark blue)
- Background gradients: Various blue/purple combinations

### Adding Photos

Replace the `.image-placeholder` elements with actual `<img>` tags:

```html
<div class="image-gallery">
    <img src="path/to/your/photo.jpg" alt="Description" />
</div>
```
